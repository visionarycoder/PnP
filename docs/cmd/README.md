# Windows Command Prompt (CMD) Snippets

Collection of useful Windows Command Prompt commands and batch scripts.

## Index

- [Basic Commands](basic-commands.md) - Essential CMD commands for file and directory operations
- [Batch Scripts](batch-scripts.md) - Batch file examples and automation

## Categories

### File & Directory Operations

- Navigation (cd, dir, tree)
- File manipulation (copy, move, del, rename)
- Directory management (mkdir, rmdir)

### System Information

- System details (systeminfo, ver, whoami)
- Process management (tasklist, taskkill)
- Environment variables (set, echo)

### Network Utilities

- Connectivity (ping, tracert, nslookup)
- Network configuration (ipconfig, netstat)
- File transfer (ftp, telnet)

### Automation & Scripting

- Batch files (.bat, .cmd)
- Conditional logic and loops
- Error handling and logging
