# Utility Snippets

Collection of cross-platform utility scripts and tools for common development tasks.

## Index

- [General Utilities](general-utilities.md) - Cross-platform utility functions including data conversion, text processing, date/time operations, and system utilities
- [Configuration Helpers](configuration-helpers.md) - Application configuration management with support for multiple formats (JSON, YAML, INI, ENV)
- [Logging Utilities](logging-utilities.md) - Comprehensive logging solutions with structured output, rotation, and analysis tools

- String manipulation and cleaning
- Regular expression utilities
- Text extraction and parsing

### Date and Time Operations

- Timestamp conversions and formatting
- Date calculations and comparisons
- Time zone handling
- Duration measurements

### Configuration Management

- Multi-format configuration file parsing
- Environment variable management
- Configuration validation and templates
- Settings inheritance and defaults

### Logging & Monitoring

- Structured logging (JSON format)
- Multiple output destinations (console, file, syslog)
- Log rotation and archiving
- Performance monitoring and analysis
- Context-aware and request logging

### System & File Utilities

- File system operations and utilities
- Directory management and cleanup
- Process monitoring and control
- Cross-platform compatibility helpers
